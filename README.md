# ExitPoll 2569

Exit Poll แบบ PWA สำหรับการเลือกตั้งสมาชิกสภาผู้แทนราษฎร 2569

เว็บไซต์โฮสต์:
<https://gkawin.github.io/exitpoll/>

## วัตถุประสงค์

- เพื่อรวบรวมข้อมูลการลงคะแนนเสียงเลือกตั้งสมาชิกสภาผู้แทนราษฎร 2569 อย่างมีประสิทธิภาพ
- เพื่อสนับสนุนการวิเคราะห์ผลการเลือกตั้งและแนวโน้มทางการเมืองในประเทศไทย
- เพื่อส่งเสริมความโปร่งใสและความน่าเชื่อถือในการเลือกตั้งในต่างประเทศ

## วิธีใช้งาน

1. เปิดเว็บไซต์ ExitPoll บนอุปกรณ์ของคุณ (คอมพิวเตอร์, แท็บเล็ต, หรือสมาร์ทโฟน)
2. เลือกพรรคการเมืองที่คุณต้องการลงคะแนนเสียง
3. หากมีสิทธิ์ลงประชามติแก้รัฐธรรมนูญ ให้เลือกตัวเลือกที่เหมาะสม
4. หลังจากเลือกเสร็จสิ้น ระบบจะบันทึกข้อมูลการลงคะแนนเสียงของคุณ
5. หลังจากการทำ Exit Poll เสร็จสิ้น Staff สามารถส่งออกข้อมูลเป็นไฟล์ CSV และ upload ไปยังเซิร์ฟเวอร์กลางเพื่อการวิเคราะห์ต่อไปได้ (กำลังพัฒนา) แต่ถ้าไม่ได้ หรือทำไม่ทัน เดี๋ยวไปใล่ขอ Download File จาก Staff ทีหลังละกัน

## วิธีการติดตั้ง (สำหรับ Staff) โดยการ add เว็บไซต์ไปยังหน้าจอหลัก (Home Screen) ของสมาร์ทโฟน

1. เปิดเว็บไซต์ ExitPoll บนอุปกรณ์สมาร์ทโฟนของคุณ
2. สำหรับ iOS (Safari):
   - แตะที่ไอคอนแชร์ (Share icon) ที่ด้านล่างของหน้าจอ
   - เลื่อนลงและเลือก "เพิ่มไปยังหน้าจอหลัก" (Add to Home Screen)
   - ตั้งชื่อแอปพลิเคชันตามที่ต้องการ แล้วแตะ "เพิ่ม" (Add)
3. สำหรับ Android (Chrome):
   - แตะที่ไอคอนเมนู (สามจุดแนวตั้ง) ที่มุมขวาบนของหน้าจอ
   - เลือก "เพิ่มไปยังหน้าจอหลัก" (Add to Home Screen)
   - ตั้งชื่อแอปพลิเคชันตามที่ต้องการ แล้วแตะ "เพิ่ม" (Add)
4. แอปพลิเคชัน ExitPoll จะถูกเพิ่มไปยังหน้าจอหลักของสมาร์ทโฟนของคุณ และสามารถใช้งานแบบออฟไลน์ได้

## การพัฒนา

ทีมงานได้นำปัญหาที่เกิดขึ้นจากการทำผล [Exit Poll เมื่อการเลือกตั้งปี 2566](https://github.com/Paulpoomrit/VancouverExitPoll2566) มาปรับปรุง เช่น

- ได้เพิ่มโลโก้พรรค และเบอร์พรรคการเมือง เพื่อให้ผู้ใช้สามารถระบุพรรคได้ง่ายขึ้น
- ลดการเกิด bias และข้อกังขาอันเกิดจากการวางพรรคใดพรรคหนึ่งไว้บนสุดของหน้่าจอตลอดเวลา ซึ่งแก้ไขโดย ทุกครั้งที่ทำ Exit Poll เสร็จสิ้น ระบบจะสลับตำแหน่งพรรคการเมืองแบบสุ่ม
- เพิ่มตัวเลือกการลงประชามติแก้รัฐธรรมนูญ เพื่อเก็บข้อมูลความคิดเห็นของผู้มีสิทธิ์เลือกตั้งในต่างประเทศ
- การลืม Run script ก่อนการเลือกตั้ง ทำให้ไม่สามารถเก็บข้อมูลได้ ทีมงานจึงได้พัฒนาเป็น Progressive Web App (PWA) เพื่อให้สามารถใช้งานแบบออฟไลน์ได้

## เทคโนโลยีที่ใช้

โครงการนี้พัฒนาโดยใช้เทคโนโลยีเว็บสมัยใหม่ เช่น HTML, CSS, JavaScript และ IndexedDB สำหรับการจัดเก็บข้อมูลแบบออฟไลน์ นอกจากนี้ยังใช้แนวทาง Progressive Web App (PWA) เพื่อให้ผู้ใช้สามารถใช้งานแอปพลิเคชันได้อย่างราบรื่นแม้ในสภาพแวดล้อมที่ไม่มีการเชื่อมต่ออินเทอร์เน็ต

โครงการนี้ใช้ AI ในการช่วยเขียนโค้ดเพื่อเพิ่มประสิทธิภาพและลดเวลาในการพัฒนาของโครงการ อย่างไรก็ตาม โค้ดทั้งหมดได้รับการตรวจสอบและปรับปรุงโดยนักพัฒนามนุษย์เพื่อให้แน่ใจว่าเป็นไปตามมาตรฐานคุณภาพและความต้องการของโครงการ

รวมถึงเอกสาร README นี้ก็ใช้ AI ช่วยร่างขึ้นมาเช่นกัน เพราะนักพัฒนามนุษย์บางทีก็ขี้เกียจเขียน :)

## ข้อสังเกต

เนื่องจากจะมีการเปลี่ยนแปลงสถานที่การเลือกตั้งในนคร Vancouver ซึ่งเรา "คาดการณ์" ว่า เราจะไม่สามารถตั้งโต๊ะ ต่อคอมพิวเตอร์ได้ เพราะอาจจะไม่มี power supply โดยตรง ดังนั้น เราจึงเปลี่ยนมาใช้วิธีการทำ Exit Poll ผ่านสมาร์ทโฟนแทน โดยให้ Staff คอยช่วยเหลือผู้ลงคะแนนเสียงในการทำ Exit Poll ผ่านเครื่องมือของ Staff แทน ซึ่งวิธีนี้ช่วยลดความยุ่งยากในการตั้งค่าอุปกรณ์และการเชื่อมต่ออินเทอร์เน็ตในสถานที่เลือกตั้ง
